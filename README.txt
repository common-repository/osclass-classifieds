Osclass classifieds software integration for Wordpress

== Description ==

Add your own classifieds ads section to your Wordpress site, quickly post and manage classifieds listings via Wordpress with [Osclass.org](http://osclass.org "Osclass").

If you are using Wordpress you can now have a classifieds section for free with Osclass technology, Open Source Classifieds script used already by over 34.000 classifieds websites all over the world.

- Publish new classifieds listings.
- User registration required.
- User account verification via email. 
- Full category list page. 
- Browse listings/ads page. 
- Search classifieds by cateogory, keyword, price range and location (country, region and city).
- Wordpress administrator, manage users, you can enable, disable and block user accounts.
- Wordpress administrator, manage listings, you can enable, disable and block listings.
- Admin notifications: new user created, new listing published. 

Install and configure your WordPress Classifieds website in less than two minutes with Osclass Classifieds Plugin. After uploading and activating the plugin, you will have a functional classifieds website ready to use.

[Terms&Conditions](https://osclass.org/page/legal-note#osclass-free)

== Installation ==

This section describes how to install the plugin and get it working.

1. Activate the plugin.
2. Setup your classifieds location (Country data) and finish setup.
3. Validate your account, an email will be send to the administrator email. 
4. You'll be able to access your Osclass Classifieds sections using your WordPress admin panel.

NOTES:
- Emails need to be working since listings need email validation. 
- You can add the "Osclass Classifieds" page to your theme menu.
